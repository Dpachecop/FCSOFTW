
# App Gestión de Equipos de Fútbol - Corte 1 ⚽📱

Bienvenido al repositorio de la **App de Gestión de Equipos**, una solución móvil diseñada para la administración técnica de clubes, desarrollada para el primer corte de la asignatura de Desarrollo de Aplicaciones Móviles.


## 🛠️ Tecnologías Utilizadas

Este proyecto utiliza herramientas de desarrollo móvil visual y componentes de hardware:

* **Kodular Creator**: Plataforma de desarrollo basada en bloques.
* **TinyDB**: Almacenamiento local persistente (Offline-first).
* **Acelerómetro**: Sensor para detección de gestos físicos.
* **Lógica de Bloques**: Arquitectura de eventos para validación y CRUD.

## 🚀 Cómo Funciona (Explicación Técnica)

1. **Persistencia (TinyDB)**: Gestión de datos mediante etiquetas dinámicas para Usuarios y Equipos.
2. **Seguridad**: Sistema de Login/Registro con validación de credenciales local.
3. **CRUD y Reportes**: Filtros parametrizados por Categoría y País.
4. **UX (Sensores)**: Función de "Shake to Clear" mediante el acelerómetro.

## 📦 Cómo Correr el Proyecto

### Opción 1: Kodular (.aia)
1. Descarga el archivo `.aia`.
2. Impórtalo en [Kodular Creator](https://creator.kodular.io/).

### Opción 2: Android (.apk)
1. Descarga el `.apk` de la carpeta `release`.
2. Instálalo en tu dispositivo permitiendo orígenes desconocidos.
