App Gestión de Equipos de Fútbol - Corte 1 ⚽📱
Descripción
Esta aplicación móvil ha sido desarrollada como parte del Corte 1 de la asignatura de Desarrollo de Aplicaciones Móviles. El objetivo principal es la gestión administrativa de equipos de fútbol, permitiendo el almacenamiento local de datos, la autenticación de usuarios y la generación de reportes específicos.

La aplicación está construida utilizando la plataforma Kodular, enfocándose en la lógica de programación por bloques y la persistencia de datos en el dispositivo.

🚀 Requisitos del Proyecto (Corte 1)
Persistencia: Uso de TinyDB para el almacenamiento de datos local (offline).

CRUD Completo: Gestión de dos tablas (Usuarios y Equipos de Fútbol).

Seguridad: Sistema de Login y Registro de usuarios.

Sensores: Implementación de sensores de hardware (Acelerómetro).

Reportes: Visualización de datos filtrados de forma parametrizada.

✨ Funcionalidades
Módulo de Seguridad:

Registro de nuevos usuarios (Nombre, Email, Usuario y Clave).

Inicio de sesión con validación de credenciales en tiempo real.

Gestión de Equipos (CRUD):

Crear: Formulario completo con 13 campos (Nombre, Eslogan, Técnico, País, Ciudad, Categoría, Goles, etc.).

Leer: Listado dinámico de todos los equipos registrados con buscador por nombre.

Actualizar: Posibilidad de modificar cualquier campo de un equipo existente manteniendo su ID único.

Eliminar: Opción de borrado de registros con confirmación de usuario.

Reportes Parametrizados:

Filtro de equipos por Categoría.

Filtro de equipos por País de origen.

Interacción por Sensores:

Acelerómetro: Al agitar el dispositivo en el formulario de creación, se limpian todos los campos automáticamente (UX mejorada).

🛠️ Instrucciones de Instalación
Para probar o modificar esta aplicación, sigue estos pasos:

Importar el código fuente:

Descarga el archivo .aia presente en este repositorio.

Ve a Kodular Creator.

Selecciona Projects > Import project (.aia) from my computer.

Instalar en Android:

Descarga el archivo .apk de la carpeta release (o genéralo desde Kodular en Export > Android App (.apk)).

Pasa el archivo a tu dispositivo móvil e instálalo (asegúrate de permitir orígenes desconocidos).
